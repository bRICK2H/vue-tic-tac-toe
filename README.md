# Проект - Tic tac toe

### Описание:
- Состояние приложения реализовано на 90% во vuex, остальное содержится в компонентах.
- Игра имеет 2 режима:
      - с компьютером (сложный уровень);
      - с самим собой;
- Реализован подсчёт очков.
- Ui свой, без сторонних плагинов и т.п.

### Инструменты разработки:
- `Vue` / `JavaScript`
- `CSS` / `SCSS`
- `HTML`
